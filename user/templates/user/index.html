図書管理(仮)とは

社内との情報共有で読書・勉強をよりよいものに

社内にある書籍を管理し、貸出可能な書籍に関しては貸出を行うことができます。

また、書籍情報の共有としてレビュー機能を実装し、自身が求めている情報を持つ本、読んだ人の感想を見てから読みたい

そういった方々に円滑に書籍を選んでいただけるようになっています。

機能説明

検索機能

書籍名での検索だけではなく、タグ名での検索機能によって、特定のジャンルでの検索が可能に

欲しい情報を持っている書籍を素早く検索することができる

オススメ書籍・新着書籍

よく貸し出しが行われている本や新着の本をメイン画面に表示しています。
